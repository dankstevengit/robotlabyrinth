This is a Unity Project (ver. 2019.1.1f1) that generates random NxN-sized maze

# How to use
First, you need to enter the size of the maze N - length and height.
After that, by pressing the left mouse button on the floor of the maze, you place a black square in the maze.
This is the finishing object, in the collision of the robot with which the end of the passage of the robot is indicated.
Then, by pressing the left mouse button on the floor of the maze, you put it on the floor.
It automatically begins to pass the maze until it collides with the finishing object.
